# 🚀 Dynamic Firebase Portfolio Template

A fast, modern, and data-driven personal portfolio template built with raw HTML, CSS, JavaScript, and powered by **Firebase Firestore**. No heavy frameworks, just pure performance and clean code.

## ✨ Key Features

* **📊 Data-Driven Content:** Projects, services, resume timeline, and skills are fetched dynamically from Firestore.
* **🌗 Dark & Light Mode:** Built-in seamless theme toggler with glassmorphism UI elements.
* **🌍 Bilingual Support:** Full support for English and Arabic (RTL) out of the box.
* **✉️ Serverless Contact Form:** Fully functional contact form using [FormSubmit](https://formsubmit.co) (No backend required).
* **📈 Analytics Ready:** Pre-configured Google Analytics 4 (GA4) custom events (e.g., CV downloads, project views, service requests).
* **🛡️ Admin Dashboard:** Includes a starting template for an Admin Panel (`admin.html`) to manage your Firestore data.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Backend/Database:** Firebase (Firestore)
* **Analytics:** Google Analytics 4 (GA4)
* **Hosting Integration:** Ready for Firebase Hosting
